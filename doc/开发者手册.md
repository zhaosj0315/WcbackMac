# 开发者手册

## 1. 项目定位

本项目是 **Mac 版微信聊天记录导出工具**，核心入口均在 `scripts/mac_*.py`。

仓库同时保留了原 Windows/PyQt5 桌面端代码（`app/`、`main.py`），但 **Mac 用户无需关注 Windows 部分**，也不需要安装 MSVC 或 Windows 构建工具。

---

## 2. 安装

```bash
# Python 3.10 / 3.11 / 3.12，不支持其他版本
git clone https://github.com/zhaosj0315/WcbackMac
cd WcbackMac
pip install -r requirements.txt
```

前置条件：macOS 12+，Xcode Command Line Tools（提供 LLDB），微信 4.x 已登录。

---

## 3. Mac 核心脚本

所有脚本从**项目根目录**运行，默认数据库路径为 `app/DataBase/MacMsg`。

### 3.1 主流程（推荐）

```bash
# 1. 探测环境
python3 scripts/mac_probe_wechat.py

# 2. 一键解密（实验性，依赖 LLDB 权限）
python3 scripts/mac_auto_decrypt_export.py --quit-original

# 3. 生成联系人映射
python3 scripts/mac_contact_mapper.py

# 4. 启动 Web 查看器
python3 scripts/mac_web_server.py
# 访问 http://127.0.0.1:5000

# 5. 全量导出
python3 scripts/mac_export_by_session.py --output data --no-global-media
```

### 3.2 已有 key 时跳过 LLDB

```bash
python3 scripts/mac_decrypt_from_keys.py \
  --keys /tmp/wechat_lldb_key_candidates.json \
  --output app/DataBase/MacMsg --verify
```

### 3.3 单会话调试

```bash
python3 scripts/mac_export_by_session.py \
  --wxid wxid_xxx --output data/test --no-global-media
```

## 4. 脚本分类

### 核心脚本（当前推荐使用）

| 脚本 | 用途 |
|------|------|
| `mac_probe_wechat.py` | 探测微信进程和数据库路径 |
| `mac_auto_decrypt_export.py` | 一键解密（实验性） |
| `mac_decrypt_from_keys.py` | 从已有 key 批量解密 |
| `mac_contact_mapper.py` | 生成联系人映射 `data/mac_contact_mapping.json` |
| `mac_export_by_session.py` | **主导出入口**，生成完整 data/ 结构 |
| `mac_web_server.py` | FastAPI Web 查看器 |
| `mac_message_utils.py` | 消息解析、媒体定位基础库（被其他脚本 import） |
| `mac_chat_analysis.py` | 聊天统计分析 |
| `mac_memoai_prepare.py` | 生成 Ollama AI 训练数据 |
| `mac_realtime_monitor.py` | 实时监听新消息 |

### 遗留脚本（功能已被主入口覆盖，当前不推荐）

`mac_export_html.py`、`mac_export_txt.py`、`mac_export_word.py`、`mac_export_json.py`、`mac_export_messages.py`、`mac_export_csv_enhanced.py`、`mac_export_wechat_style_html.py`、`mac_export_all.py`、`mac_export_media.py`、`mac_export_sns.py`、`mac_export_favorite.py`

这些脚本仍可独立运行，但 `mac_export_by_session.py` 已覆盖其全部功能。

## 6. 关键技术点

- **消息解码**：Mac 微信 4.x 用 zstd 压缩，格式 `wxid:\n<内容>`，需先解压再解析
- **数据库分片**：`message_0.db`（最新）到 `message_9.db`（最旧），每个分片有独立 `Name2Id` 表
- **发送者识别**：`real_sender_id` 是分片内局部 rowid，必须按分片查 `Name2Id`，不能跨分片混用
- **媒体定位**：先查 `MessageTemp/<会话md5>/`，找不到按 `create_time` 降级匹配，再兜底查 `attach/`
- **my_wxid 检测**：自动从 `xwechat_files/wxid_*` 目录名推断，也可 `--my-wxid` 手动指定

## 7. 数据库查看

用任意 SQLite 工具打开 `app/DataBase/MacMsg/` 下的 `.db` 文件即可浏览原始数据。

推荐工具：[DB Browser for SQLite](https://sqlitebrowser.org/dl/)
